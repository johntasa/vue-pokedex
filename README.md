# Pokedex

## Tech stack

- **Vite (Build)**
- **Vue 3 (Composition API)** 
- **Pinia (Store)**
- **Vue Router**
- **Jest (Unit Tests)** 
- **Vercel (Deployment)**

Deployed at: https://vue-pokedex-filter.vercel.app/

### Version

- **node:** 22.13.1
- **npm:** 10.9.2

## Install & run

```bash
npm install
npm run dev
```
